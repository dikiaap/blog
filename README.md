# Diki Ananta's blog site

[![Status Build](https://img.shields.io/travis/com/dikiaap/blog.dikiaap.id/gh-pages.svg)](https://travis-ci.com/dikiaap/blog.dikiaap.id)

## Requirements

- Ruby
- RubyGems

## Build

```shell
gem install jekyll jekyll-paginate jekyll-sitemap
jekyll build
jekyll serve
```
